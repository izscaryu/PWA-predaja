# PWA predaja

Ovo je sanitizirana verzija projekta pripremljena za javni repozitorij.

- `calisthenicshr_public.sql` sadrži samo shemu baze (bez korisničkih podataka).
- Originalna `calisthenicshr.sql` je osjetljiva i izuzeta iz repozitorija.

Upute za lokalni razvoj:
1. Ako imate privatni dump `calisthenicshr.sql`, uvezite ga lokalno u MySQL.
2. Konfigurirajte `connect.php` s lokalnim podacima za pristup bazi (ne stavljajte lozinke u repozitorij).
